# 나눔스퀘어 폰트 배치 위치

이 폴더에 **나눔스퀘어(NanumSquare)** 폰트 파일(`.ttf` 또는 `.otf`)을 넣으면
앱 시작 시 런타임에 자동 등록되어 모든 메뉴·UI에 적용됩니다.

## 권장 파일
아래 4종을 넣으면 굵기(Light/Regular/Bold/ExtraBold)가 모두 매핑됩니다.

```
NanumSquareL.ttf    (Light)
NanumSquareR.ttf    (Regular)
NanumSquareB.ttf    (Bold)
NanumSquareEB.ttf   (ExtraBold)
```

## 받는 곳
- 네이버 나눔글꼴: https://hangeul.naver.com/font
- (라이선스: 무료 배포, OFL 유사 — 재배포 조건 확인 후 커밋 권장)

> 폰트가 시스템에 이미 설치되어 있으면 이 폴더가 비어 있어도 자동으로 사용됩니다.
> 둘 다 없으면 시스템 폰트로 안전하게 폴백합니다.
